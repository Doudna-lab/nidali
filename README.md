## CLONING THIS REPOSITORY

<details>
<summary>Standard Git</summary>
<ul>

  Clone repository files
```
git clone https://github.com/Doudna-lab/nidali.git
```
</details>

<details>
<summary>Git LFS</summary>
<ul>
  
  Linux Install
```
apt install git-lfs
git lfs install
```

  macOS Install
```
brew install git-lfs
git lfs install
```

  Pull large files
```
git lfs pull
```
</details>
